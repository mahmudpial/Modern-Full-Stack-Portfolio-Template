1. Go to https://github.com/new
2. Fill in details:
   ```

   Description: Modern Full-Stack Portfolio Template by Pial Mahmud
   Public: ✅ YES (for better marketing)
   Initialize with README: ❌ NO (we have one)
   Add .gitignore: ❌ NO (we have one)
   ```
3. Click "Create repository"
4. Copy the HTTPS URL (you'll need it next)

### Step 2️⃣: Configure Git Locally

Open terminal in your project folder:

```bash
# Navigate to project
cd ~/Documents/new\ websites/web-product-for-sell/protfolio_templete

# Initialize Git (if not already done)
git init
git add .
git commit -m "Initial commit: Premium Portfolio Template"
git branch -M main
```

### Step 3️⃣: Connect to GitHub

```bash
# Replace YOUR_USERNAME with your actual GitHub username
git remote add origin https://github.com/YOUR_USERNAME/portfolio-template.git
git push -u origin main
```

✅ **Done!** Your repository is now on GitHub!

## 🔐 PROTECT YOUR WORK

### What the License Does

Your **CC BY-NC-ND 4.0** license:

- ✅ Lets people use it freely (non-commercial)
- ✅ Requires them to give you credit
- ❌ Prevents commercial use without buying license
- ❌ Prevents modification without buying license
- ❌ Prevents redistribution

### Monitor for Violations

Use Google Alerts:

```
1. Go to: https://alerts.google.com
2. Create alert: "portfolio-template pial mahmud"
3. Create alert: "portfolio-template" + "gumroad"
4. Get notified of mentions/violations
```

### Handle Violations

If someone uses your template commercially:

1. **First:** Send friendly reminder (often people don't know)
2. **Second:** Formal DMCA takedown notice
3. **Third:** Legal action (rarely needed)

**DMCA Template:**

```
Subject: DMCA Takedown Notice - Unauthorized Commercial Use
```
